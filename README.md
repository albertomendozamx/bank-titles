# UBTM

User's bank titles manager

## Requirements

- Node ^16

## Environment variables

```
- No one
```

## How to install?

```bash
$ yarn install
```

## How to run test?

```bash
$ yarn test
```

## How to compile it?

```bash
$ yarn build
```

## How to run it on staging?

```bash
$ yarn dev
```

## How to run it on production?

```bash
$ node build/src/index.js
```
